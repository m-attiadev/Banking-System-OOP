# 🏦 Banking System Management

نظام متكامل لإدارة البنك مكتوب بـ **C++**

---

## 📋 **الوصف العام**

نظام بنكي احترافي يوفر إدارة شاملة للعملاء والمستخدمين والمعاملات البنكية بواجهة رسومية نصية (Console UI) منظمة وسهلة الاستخدام.

---

## ✨ **المميزات الرئيسية**

### 1️⃣ **نظام التسجيل الدخول المتقدم**
- ✅ تحقق آمن من اسم المستخدم وكلمة المرور
- ✅ حد أقصى 3 محاولات تسجيل دخول فاشلة
- ✅ سجل تاريخي لكل محاولات تسجيل الدخول
- ✅ رسائل خطأ واضحة وملونة

### 2️⃣ **إدارة العملاء (Client Management)**
- 📊 عرض قائمة جميع العملاء
- ➕ إضافة عميل جديد
- ❌ حذف عميل
- ✏️ تحديث بيانات العميل
- 🔍 البحث عن عميل معين
- 💾 حفظ البيانات في ملفات (Clients.txt)

### 3️⃣ **المعاملات البنكية (Banking Transactions)**
- 💵 **الإيداع (Deposit)** - إيداع مبلغ في حسابك
- 🏦 **السحب (Withdraw)** - سحب مبلغ من الحساب
- 📤 **التحويل (Transfer)** - تحويل أموال بين الحسابات
- 📊 **إجمالي الأرصدة** - عرض أرصدة جميع العملاء

### 4️⃣ **إدارة المستخدمين (User Management)**
- 👥 عرض قائمة المستخدمين
- ➕ إضافة مستخدم جديد
- ❌ حذف مستخدم
- ✏️ تحديث بيانات المستخدم
- 🔍 البحث عن مستخدم

### 5️⃣ **نظام الصلاحيات (Permissions System)**
```
✅ صلاحيات مختلفة:
- pListClients (عرض العملاء)
- pAddNewClient (إضافة عميل)
- pDeleteClient (حذف عميل)
- pUpdateClient (تحديث العميل)
- pFindClient (البحث عن عميل)
- pTransactionsMenue (المعاملات)
- pManageUsers (إدارة المستخدمين)
- pLoginRegister (عرض سجل الدخول)
- eAll (جميع الصلاحيات)
```

### 6️⃣ **سجل تسجيل الدخول (Login Register)**
- 📅 تاريخ ووقت كل تسجيل دخول
- 👤 اسم المستخدم
- 🔐 كلمة المرور
- 🔑 الصلاحيات المستخدمة
- 📁 حفظ في ملف (LoginHistory.txt)

---

## 🏗️ **البنية المعمارية (Architecture)**

### **نمط التصميم: OOP (Object-Oriented Programming)**

```
Main Loop (Project11OOP.cpp)
    ↓
LoginScreen (تسجيل الدخول)
    ↓
MainScreen (الشاشة الرئيسية)
    ├── ClientManagement (إدارة العملاء)
    ├── TransactionManagement (المعاملات)
    ├── UserManagement (إدارة المستخدمين)
    └── LoginRegister (سجل الدخول)
```

### **الفئات الرئيسية:**
- `clsPerson` - الفئة الأساسية للشخص
- `clsBankClient` - بيانات العميل البنكي
- `clsUser` - بيانات المستخدم والصلاحيات
- `clsScreen` - الفئة الأساسية للشاشات
- `clsInputValidate` - التحقق من صحة المدخلات
- `clsUtility` - الدوال المساعدة والألوان
